Flexbox Grid

BF includes Flexbox Grid as part of the core! If you're familiar with the Bootstrap grid, this will be easy for you to use!

Markup

<div class="container">
  <div class="row">
    <div class="col-xs-12 col-sm-12 col-md-12 col-lg-12">
    ...
</div></div></div>

NOTE: The outer container class is optional.

.col-xs-12 - 12 columns (full-width) on extra small device
.col-sm-6 - 6 columns (half-width) on small device
.col-md-4 - 4 columns (one-third width) on tablets
.col-lg-3 - 3 columns (one-fourth width) on desktops

Container Classes
.container - standard width
.container-fluid - full-width (100%)
.container-1200 - 1200px max width
.container-960 - 960px max width

B3Grid

Brutal Blocks 'n Bricks Grid

A grid for when you don't want rows! B3Grid allows you to create a flex layout without being restricted by rows. This is ideal for long lists of repeating content blocks. B3Grid can be used within a Flexbox grid layout, or independently.

Blocks vs. Bricks

Blocks = flexible blocks that evenly fill up the entire width of an area.

Bricks = inline blocks that don't evenly fill up an area's width.

.block-wrap - blocks outer wrapper
.block - individual blocks

.brick-wrap - bricks outer wrapper
.brick - individual bricks

Basic Markup

Create a 3-column flex layout:

<div class="block-wrap">
  <div class="block">...</div>
  <div class="block">...</div>
  <div class="block">...</div>
</div>

Create 3 inline bricks:

<div class="brick-wrap">
  <div class="brick">...</div>
  <div class="brick">...</div>
  <div class="brick">...</div>
</div>

Quick Widths

Quick Widths are a simple way to evenly divide up the widths of blocks or bricks. Instead of assigning a specific width to each individual block or brick, you can just apply a class to the block- or brick-wrap container.

Markup
<div class="brick-wrap thirds">
  <div class="brick">...</div>
  <div class="brick">...</div>
  <div class="brick">...</div>
</div>

Classes
.full - 100% width
.halves - 1/2 (50%) width
.thirds - 1/3 (33%) width
.quarters - 1/4 (25%) width
.fifths - 1/5 (20%) width
.sixths - 1/6 (17%) width

Nesting

Blocks and bricks can be nested within each other. In this example, we'll add 3 bricks set to a one-third width each, nested within a two-thirds width block.

<div class="block-wrap">
 <div class="block b66">
    <div class="brick-wrap">
      <div class="brick b33">...</div>
      <div class="brick b33">...</div>
      <div class="brick b33">...</div>
    </div>
 </div>
 <div class="block b33">...</div>
</div>

Tiles

Tiles are a simple way to make unordered lists into evenly divided inline blocks. Simply apply class tiles to any ul tag! By default, li tile items maxes out at 50% width. Tiles self-stretch to be the same height as other tiles on the same line.

Markup
<ul class="tiles thirds">
  <li>  ...  </li>
  <li>  ...  </li>
  <li>  ...  </li>
</ul>

Quick-Width Classes
.halves - 1/2 (50%) li width
.thirds - 1/3 (33%) li width
.quarters - 1/4 (25%) li width
.fifths - 1/5 (20%) li width
.sixths - 1/6 (17%) li width

Pad Tiles
Tiles have no padding by default. Add padding to all tiles automatically by applying the class .padded to a tiles wrapper:
ul.tiles.padded

CSS Grids

As an alternate method for creating layouts, we've included support for CSS grids. By default, items are inserted by row and have no gap.

Grid
.grid - REQUIRED wrapper for CSS grid layout
.columns - insert items by column
.gap - adds a 1em spacing gap between items

Column Widths
.col2 - 2-column layout
.col3 - 3-column layout
.col4 - 4-column layout
.col5 - 5-column layout
.col6 - 6-column layout

Row Heights
.rh25 - one quarter height of screen
.rh33 - one-third height of screen
.rh50 - half the height of screen
.rh66 - two-thirds height of screen
.rh75 - three-fourths height of screen
.rh100 - full height of screen

Example Markup

The below code will create a 3-column layout with rows occupying one-third of the height of the screen, with gap spacing added between items.

<div class="grid col3 rh33 gap">
  <div>  Item 1  </div>
  <div>  Item 2  </div>
  <div>  Item 3  </div>
</div>

Item Column Span
Span individual items across columns:
.span2 - span 2 columns
.span3 - span 3 columns
.span4 - span 4 columns
.span5 - span 5 columns

Item Row Span
Span individual items across rows:
.r-span2 - span 2 rows
.r-span3 - span 3 rows
.r-span4 - span 4 rows
.r-span5 - span 5 rows

Item Start
Determine which column the first item will start on:
.start2 - start on 2nd column
.start3 - start on 3rd column
.start4 - start on 4th column
.start5 - start on 5th column

Example Markup

The below code will create a 4-column layout, with the first item spanning two columns and starting on the 2nd colun.

<div class="grid col4">
  <div class="start2 span2">  Item 1  </div>
  <div>  Item 2  </div>
  <div>  Item 3  </div>
  <div>  Item 4  </div>
  <div>  Item 5  </div>
</div>
