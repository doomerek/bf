Getting Started

Most browsers support some basic symbols, with no dependencies. bIcons allows you to render them by applying classes! Keep in mind that various browsers may render symbols differently. Simply use the following markup, where "icon-name" is any of the below class names.

<i class="bi bi-icon-name"></>
